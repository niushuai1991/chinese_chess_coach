# 中国象棋完整规则

## 一、棋盘与棋子

### 棋盘规格
- **纵线**：9条
- **横线**：10条
- **交叉点**：90个（棋子放置位置）
- **河界**：第5行与第6行之间，标有"楚河汉界"
- **九宫**：双方各一个（第0-2行第3-5列，第7-9行第3-5列），有斜线

### 棋子配置
每方16枚棋子，共32枚：

| 棋子 | 数量 | 初始位置 |
|------|------|----------|
| 将/帅 | 1 | 九宫中心（底行第4列） |
| 士/仕 | 2 | 九宫内（底行第3、5列） |
| 象/相 | 2 | 第3行第2、6列 |
| 马/傌 | 2 | 第3行第1、7列 |
| 车/俥 | 2 | 底行第0、8列 |
| 炮/炮 | 2 | 第3行第1、7列 |
| 卒/兵 | 5 | 第4行所有列 |

## 二、各棋子走法

### 1. 将/帅
**活动范围**：己方九宫内（3×3区域）

**走法**：
- 每次可在九宫内**直行或横行一步**
- **不能走出九宫**
- **不能与对方将帅在同一条纵线上面对面**（中间无棋子阻挡），称为"将帅对脸"或"老将照面"

**胜负**：将/帅被将死则输

### 2. 士/仕
**活动范围**：己方九宫内

**走法**：
- 每次可在九宫内**斜行一步**
- **不能走出九宫**

### 3. 象/相
**活动范围**：己方区域，**不能过河**

**走法**：
- **象行田**：每次**斜走两步**（沿对角线移动两个交叉点）
- **不能过河**
- **塞象眼**：如果"田"字中心有棋子，则不能走该方向
- 象主要用于防守

### 4. 马/傌
**活动范围**：全棋盘

**走法**：
- **马行日**：先**直走一步**，再**斜走一步**（或理解为先横走一步再竖走一步）
- 可向8个方向移动
- **蹩马腿**：如果直行方向紧邻处有棋子（无论敌我），则不能往该方向移动
- 马不因被蹩腿而受其他方向限制

**示例**：马在(3,3)可跳到：
- (1,2)、(1,4)、(2,1)、(2,5)、(4,1)、(4,5)、(5,2)、(5,4)

### 5. 车/俥
**活动范围**：全棋盘

**走法**：
- **直行或横行不限距离**
- **不能跨越棋子**
- 路径上无棋子阻挡时可到达任意位置
- 是威力最大的棋子之一

### 6. 炮/炮
**活动范围**：全棋盘

**走法**：
- **不吃子时**：走法与车相同（直行或横行，不限距离）
- **吃子时**：必须**跳过一个棋子**（炮架/炮台），无论该棋子是敌我双方
- 炮架与目标之间不能有其他棋子
- 路径上无棋子时可移动到任意位置

### 7. 卒/兵
**活动范围**：全棋盘

**走法**：
- **过河前**（未过河界）：每次只能**向前直行一步**
- **过河后**：可以**向前、向左、向右横行一步**，每次一步
- **永远不能后退**
- 过河后威力增强

## 三、胜负与和棋规则

### 胜负判定
1. **将死**：一方的将/帅被将军且无法解除将军状态，则判负
2. **困毙**：轮到一方走棋时无子可动（不算将死情况），则判负
3. **认输**：一方主动认输
4. **超时**：限时比赛中超时判负

### 将军（照将）
- 一方的棋子攻击对方的将/帅，下一步能吃掉将/帅，称为"将军"或"照将"
- 被将军方必须：
  1. 用棋子挡住攻击（垫子）
  2. 移动将/帅避开攻击
  3. 吃掉攻击的棋子
  
- 若无法解除将军，则为"将死"，判负

### 和棋情况
1. **双方同意和棋**
2. **双方无子可动**（困毙除外）
3. **循环重复局面**：同一局面重复三次或以上
4. **六十回合规则**：双方连续60回合未吃子
5. **双方子力不足**：无法将死对方（例如：
   - 单帅对单帅
   - 单帅对单士/单象
   - 单马/单炮/单兵对单帅
   等）

### 禁止规则
1. **长将**：连续将军三次以上，可判定为和棋
2. **长捉**：连续捉对方同一棋子多次
3. **长杀**：连续做杀
4. **二打一还**：双方循环重复的攻击与反击

## 四、特殊规则

### 王不照面（将帅不能对面）
- 双方将帅不能在同一条纵线上直接相对
- 中间必须至少有一个棋子
- 任何走法不能导致己方将帅与对方将帅直接相对

### 楚河汉界规则
- **象/相**不能过河
- **卒/兵**过河后能力增强（可横行）
- **其他棋子**可自由过河

### 九宫规则
- **将/帅、士/仕**只能在九宫内活动
- 九宫有斜线供士/仕斜行
- 九宫大小为3×3（9个交叉点）

## 五、对局流程

### 开局
- 红方先走（由执红方先行）
- 双方轮流走棋
- 开局阶段注重出子和布局

### 中局
- 双方主力交锋
- 战术组合与攻防转换
- 兑子与突破

### 残局
- 棋子较少
- 兵/卒价值上升
- 车/马/炮的配合应用

## 六、记谱方法（简要）

### 中文记谱法
- 纵线：红方从右到左用"一"到"九"，黑方从右到左用"1"到"9"
- 棋步：棋子名称+纵线+动作+位置
  - "炮二平五"：炮从二线平移到五线
  - "马八进七"：马从八线进到七线

## 七、基本战术

### 常见杀法
1. **闷杀**：将/帅被困死
2. **双车错**：双车配合杀棋
3. **卧槽马**：马跳到九宫前位置配合车炮
4. **挂角马**：马在九宫角配合车炮
5. **铁门栓**：车在底线配合炮杀棋
6. **大胆穿心**：弃车/炮突破中路杀棋
7. **海底捞月**：炮在底线配合车杀棋

### 防守要点
1. 保护将/帅
2. 注意解除将军
3. 保持子力协调
4. 控制关键线路

---

**注意**：本规则遵循中国象棋协会最新竞赛规则，用于指导AI对弈系统实现。
